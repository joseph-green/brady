Introduction

BRADY is an API that provides information about the attractiveness of current NFL players, as well as other information such as team and position.

The "attractiveness" of players is determined by the Hakstack.ai API.

BRADY was created by Joseph Green as a side project.

Endpoints

Player by ID

Get a player by their player ID

ENDPOINT

GET [url]/players/<int: player id>

SAMPLE REQUEST

GET [url]/players/228

SAMPLE RESPONSE

{
 "player_id": 228,
 "player_first_name": "Tom",
 "player_last_name": "Brady",
 "player_position": "QB",
 "player_team": "New England Patriots",
 "player_attractiveness": null,
 "img_path": "../assets/player_headshots/QB_Tom_Brady.jpg"
}

Player Search by Name

Search for a player by First and Last Name

ENDPOINT
