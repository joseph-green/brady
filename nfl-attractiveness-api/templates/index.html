Introduction

[API Name] is an api that provides information about the attractiveness of current NFL players, as well as other information such as team and position.

The "attractiveness" of players is determined by the Hakstack.ai API.

The [API name] was created by Joseph Green as a side project.

Endpoints

Player by ID

Get a player by their player ID

ENDPOINT
